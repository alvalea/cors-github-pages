# Install nvm #

```
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | zsh
```

# Install node #

```
nvm ls
nvm install node
nvm ls
```

# Install Angular CLI #

```
npm install -g @angular/cli
```

# Create ng project #

```
ng new ng-example
```

# Install karma firefox launcher #

```
npm install karma-firefox-launcher
```
